How to Read the CSV File

Tastytrade

Export your trade history directly from the desktop platform.

  1. Go to the History tab.
  2. Click CSV in the top right.
  3. Ensure all columns are included (Symbol, Type, Date, Price, etc.).
  4. The auditor expects the standard Tastytrade format including "Action", "Symbol", "Value", "Commissions", "Fees".

Interactive Brokers

Create a Flex Query in Account Management.

  1. Select Trades as the section.
  2. Include columns: Symbol, Date/Time, Quantity, Price, Proceeds, Comm/Fee, Asset Class.
  3. Ensure "Executions" are detailed.
  4. Download the report as CSV.
